FastTiles tiling puzzle solver finds solutions for the puzzle using branch and bound algorithm, which in other words means backtracking algorithm.

Directory Description:
	
FastTiles: conatains the source code of this project
|--- icons: contains icon used from FamFam SILK icon set http://www.famfamfam.com/lab/icons/silk/
Puzzle Report: contains the input puzzles from Prof. Gabriel Robins website http://www.cs.virginia.edu/~robins/puzzles/ and execution time report for generating solutions for those puzzles.
